# Библиотека репозиториев GitHub

Этот раздел содержит отобранные инструменты и библиотеки для различных задач.

| Категория | Описание |
| :--- | :--- |
| [🐍 Python-Dev](./Python-Dev/) | Библиотеки, фреймворки и инструменты для Python разработки |
| [🌐 Web-Front](./Web-Front/) | Vanilla компоненты, стили и фронтенд инструменты |
| [🔗 Backend-Database](./Backend-Database/) | Облачные бэкенды, базы данных и API |
| [💬 Messaging-Bots](./Messaging-Bots/) | Мессенджеры, боты и инструменты общения |
| [🌐 Network-Proxy](./Network-Proxy/) | Сетевые инструменты, прокси и безопасность |
| [📚 Learning-Education](./Learning-Education/) | Материалы для обучения, туториалы и гайды |
| [🤖 Automation-AI](./Automation-AI/) | Инструменты автоматизации, MCP сервера и ИИ-утилиты |
| [🧠 Knowledge-Management](./Knowledge-Management/) | Управление знаниями, заметки и личная продуктивность |
| [🎬 Media-Processing](./Media-Processing/) | Обработка видео и аудио, конвертация и монтаж |
| [📂 Media-Library](./Media-Library/) | Хранение, каталогизация и бэкап медиа-архивов |
| [🛠️ Utilities](./Utilities/) | Системные тулзы, CLI и полезные скрипты |
