## Описание
Открытая альтернатива Firebase. Предоставляет полноценную базу данных PostgreSQL, аутентификацию пользователей, API в реальном времени, Edge Functions и хранилище файлов (Storage).

## Почему это полезно
- **Скорость разработки**: Позволяет развернуть полноценный бэкенд для веб-сайта или приложения за считанные минуты без необходимости писать сложный серверный код.
- **PostgreSQL**: Вы получаете всю мощь реляционной базы данных с возможностью выполнения SQL-запросов.
- **Python SDK**: Отлично интегрируется с Python, что позволяет использовать Supabase как облачное хранилище данных для ваших скриптов автоматизации.
- **Auth**: Готовая система регистрации и логина пользователей "из коробки".

## Основные возможности
- **Database**: Postgres база данных с интерфейсом управления.
- **Authentication**: Управление пользователями и правами доступа.
- **Storage**: Хранение и раздача медиа-файлов (изображения, видео).
- **Edge Functions**: Выполнение серверного кода (TypeScript) ближе к пользователю.
