# Immich

- **Теги**: #photos #videos #self-hosted #backup #google-photos-alternative

## Описание
Высокопроизводительное self-hosted решение для резервного копирования и управления фотографиями и видео. Позиционируется как современная и быстрая альтернатива Google Photos.

## Почему это стоит использовать
- **Полный контроль**: Ваши медиа-файлы хранятся на вашем сервере, а не в облаке корпораций.
- **Функциональность**: Распознавание лиц, поиск по объектам, карта фотографий и поддержка множества пользователей.
- **Мобильное приложение**: Нативные приложения для iOS и Android с функцией автозагрузки в фоне.
- **Производительность**: Очень быстрая работа интерфейса даже при огромных библиотеках (сотни тысяч файлов).

## Контекст для работы
Идеально подходит для организации личного архива исходников и готовых работ, обеспечивая удобный доступ и поиск без передачи данных третьим лицам.
