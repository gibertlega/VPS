## Описание
Современный и полностью асинхронный фреймворк для Telegram Bot API на Python. Он построен на базе `asyncio` и `pydantic`, обеспечивая высокую производительность и удобство разработки.

## Почему это стоит использовать
- **Асинхронность**: Позволяет боту обрабатывать тысячи запросов одновременно без задержек.
- **Типизация**: Глубокая интеграция с Type Hinting и Pydantic делает код чистым, понятным и защищенным от ошибок.
- **FSM (Finite State Machine)**: Встроенная поддержка состояний (конечных автоматов), что критично для создания сложных диалогов и многошаговых анкет.
- **Middlewares**: Мощная система промежуточного ПО для обработки входящих данных (логирование, аутентификация, троттлинг).

## Контекст использования
Стандарт индустрии для написания ботов на Python. Если планируете создание Telegram-бота с нуля — это лучший выбор по функциональности и поддержке сообщества.
