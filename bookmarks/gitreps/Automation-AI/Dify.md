## Описание
Dify — это платформа для разработки LLM-приложений с открытым исходным кодом. Она объединяет в себе управление моделями, создание RAG-систем (работа с базой знаний), оркестрацию агентов и визуальное проектирование рабочих процессов.

## Почему это полезно
- **Визуальный редактор**: Позволяет собирать сложные ИИ-приложения из блоков без написания кода (Low-code).
- **RAG Engine**: Мощные инструменты для индексации документов (PDF, Markdown, Notion), которые позволяют нейросетям отвечать на вопросы по вашим данным.
- **Поддержка множества моделей**: Интеграция с OpenAI, Claude, а также с локальными моделями через Ollama.
- **Enterprise-ready**: Встроенные инструменты для мониторинга, отладки и быстрого развертывания готовых приложений через API или веб-интерфейс.

## Контекст использования
Это "комбайн" для создания серьезных ИИ-инструментов. Если вам нужно быстро собрать рабочее приложение, которое умеет читать ваши документы и выполнять задачи через нейросети, Dify — лучший выбор.
