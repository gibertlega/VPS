# FFmpeg

- **URL**: [https://github.com/FFmpeg/FFmpeg](https://github.com/FFmpeg/FFmpeg)
- **Категория**: Media-Processing
- **Теги**: #video #audio #transcoding #streaming #multimedia

## Описание
Ведущий мультимедийный фреймворк, способный декодировать, кодировать, транскодировать, микшировать, фильтровать и воспроизводить практически всё, что создали люди и машины.

## Почему это стоит знать (особенно видеомонтажеру)
- **Универсальность**: Поддержка практически всех существующих кодеков и контейнеров.
- **Мощность**: Возможность автоматизировать пакетную обработку видео, извлечение аудио, создание превью и изменение метаданных без использования GUI.
- **Интеграция**: Основа для многих видеоредакторов и плееров. В Python отлично работает через библиотеку `ffmpeg-python`.

## Полезные примеры
- `ffmpeg -i input.mp4 output.webm` — простая конвертация.
- `ffmpeg -i input.mp4 -vn -ab 192k output.mp3` — извлечение аудио в высоком качестве.
- `ffmpeg -i input.mp4 -vf "scale=1280:-1" output_720p.mp4` — изменение разрешения.
