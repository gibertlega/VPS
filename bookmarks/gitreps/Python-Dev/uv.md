## Описание
Чрезвычайно быстрый менеджер пакетов и установщик для Python, написанный на Rust. Он предназначен для замены `pip`, `pip-tools` и `virtualenv`, работая в 10–100 раз быстрее аналогов.

## Почему это стоит использовать
- **Скорость**: Установка зависимостей происходит практически мгновенно благодаря агрессивному кешированию и многопоточности.
- **Единый инструмент**: Умеет управлять версиями Python, создавать виртуальные окружения и управлять зависимостями проекта.
- **Совместимость**: Полностью поддерживает стандарты Python (pyproject.toml) и является "прозрачной" заменой для привычных инструментов.
- **Надежность**: Строгое разрешение зависимостей и работа без глобальной установки пакетов минимизируют конфликты в системе.

## Контекст использования
Рекомендуется использовать во всех новых Python-проектах для ускорения разработки и автоматизации настройки окружения.
