# 🚀 Скилл: Планерка (VPS Audit & Recovery)

Этот скилл активируется, когда пользователь произносит ключевое слово **"планерка"**.

## 1. Фаза активации (Trigger: "планерка")
При получении команды, агент обязан:
1. Подтвердить готовность к полному аудиту VPS.
2. Сформировать таблицу/список всех критических процессов и скриптов, которые будут проверены.
3. Составить план команд (SSH и локальных) для детальной проверки.

### Объекты проверки (Inventory):
| Группа | Объекты |
| :--- | :--- |
| **VPN / Proxy** | `x-ui` (panel), `mtprotoproxy`, `nginx` |
| **Боты** | `lega_vps_bot` |
| **Сетевые ресурсы** | Пинг доменов `lega-vps.mooo.com`, `sub.lega-vps.mooo.com` |
| **Скрипты** | `scripts/verify_setup.py`, `scripts/backup_golden.py`, `scripts/deploy_monitor.py` |

---

## 2. Фаза выполнения (Trigger: "тест всего")
После подтверждения пользователем плана, агент переходит к активным действиям:

### Шаг 1: Сбор статусов (Health Check)
- Выполнение `systemctl status` для всех сервисов через SSH (секреты в `.env`).
- Проверка доступности портов (`443`, `8888`, `9999`).
- Запуск локального скрипта `verify_setup.py`.

### Шаг 2: Анализ логов
- Чтение `journalctl` на наличие ошибок (Critical/Error).
- Проверка логов `soft/v2rayN/guiLogs/`.

### Шаг 3: Исправление проблем (Fixing)
- Если сервис упал — попытка `systemctl restart`.
- Если файлы конфигурации повреждены — восстановление:
    - **Local**: Из папки `backup/`.
    - **Remote**: Откат через Git (`https://github.com/gibertlega/VPS.git`).
- Использование специализированных скриптов: `scripts/total_rescue.py` или `scripts/emergency_recovery.py`.
- Обновление SSL сертификатов, если срок истекает.

---

## 3. Фаза синхронизации (Final Cleanup)
Когда статус всех систем подтвержден как **"Healthy"**, агент обязан предложить следующие действия:
1. **Золотой бэкап**: Запуск `python scripts/backup_golden.py` для вытягивания текущих конфигов с сервера в локальную папку `backup/`.
2. **Git Push**: Фиксация всех изменений и бэкапов в репозитории: `git add .`, `git commit`, `git push origin master`.
