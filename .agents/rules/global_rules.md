# Правила разработки и автоматизации

## Python
- Стиль кода: PEP8.
- Использование `venv` для зависимостей.
- Типизация данных (Type Hinting) обязательна для публичных функций.

## Веб-разработка
- Core: Только HTML, Vanilla JS, Vanilla CSS.
- Без TailwindCSS, если не указано иное.
- Адаптивность и Premium Design — приоритет.

## Автоматизация
- Все скрипты в папке `scripts/`.
- Использование `run_command` для выполнения рутинных задач.
- Логирование важных этапов выполнения.

## Работа с базой знаний
- Не хранить всё содержимое `bookmarks/` в оперативной памяти.
- Обращаться к файлам заметок и репозиториев (`bookmarks/gitreps/`) только по явному запросу или при необходимости уточнить детали.
- Treat bookmarks as an external reference book, not active context.

## Git
- Push в удаленный репозиторий выполнять только по прямой команде или при риске переполнения памяти/контекста.
